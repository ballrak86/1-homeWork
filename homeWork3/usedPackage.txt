gcc (GNU Compiler Collection) - набор компиляторов для различных языков программирования. Он используется как стандартный компилятор для свободных UNIX-подобных операционных систем.
kernel-devel - Это набор хедеров, которые нужны для сборки модулей ядра
make -  утилита, автоматизирующая процесс преобразования файлов из одной формы в другую. Чаще всего это компиляция исходного кода в объектные файлы и последующая компоновка в исполняемые файлы или библиотеки. 
wget - программа для загрузки файлов по сети
kernel-headers - нужен для компиляции программ/библиотек плотно работающих с ядром, точнее работающих в некоторых случаях с ядром не через glibc
