gcc (GNU Compiler Collection) - набор компиляторов для различных языков программирования. Он используется как стандартный компилятор для свободных UNIX-подобных операционных систем.
kernel-devel - Это набор хедеров, которые нужны для сборки модулей ядра
make -  утилита, автоматизирующая процесс преобразования файлов из одной формы в другую. Чаще всего это компиляция исходного кода в объектные файлы и последующая компоновка в исполняемые файлы или библиотеки. 
wget - программа для загрузки файлов по сети
kernel-headers - нужен для компиляции программ/библиотек плотно работающих с ядром, точнее работающих в некоторых случаях с ядром не через glibc
ncurses-devel - библиотека, написанная на языках Си и Ада, предназначенная для управления вводом-выводом на терминал, в числе прочего, библиотека позволяет задавать экранные координаты и цвет выводимых символов. Предоставляет программисту уровень абстракции, позволяющий не беспокоиться об аппаратных различиях терминалов и писать переносимый код. 
openssl-devel -  полноценная криптографическая библиотека с открытым исходным кодом, широко известна из-за расширения SSL/TLS, используемого в веб-протоколе HTTPS.
flex - инструмент для генерации программ, распознающих заданные образцы в тексте. Распознавание образцов часто используется во многих приложениях. Пользователь задает правила поиска, и flex создает программу для поиска образцов. Причина, по которой люди используют flex, состоит в том, что гораздо проще задать правила, чем писать собственную программу.
bison - это GNU-выпуск известной программы YACC, предназначенной для порождения компиляторов по описанной пользователем КС-грамматике.
bzip2 - бесплатная свободная утилита командной строки с открытым исходным кодом для сжатия данных, реализация алгоритма Барроуза — Уилера.
elfutils-libelf-devel - содержит библиотеки для создания приложений для обработки скомпилированных объектов. libelf позволяет вам получить доступ к внутренним элементам формата объектного файла ELF, чтобы вы могли видеть различные разделы файла ELF.
libssl-dev - Этот пакет является частью реализации проекта OpenSSL криптографических протоколов SSL и TLS для безопасной связи через Интернет. Он содержит библиотеки разработки, заголовочные файлы и справочные страницы для libssl и libcrypto.
build-essential - пакет включает в себя коллекцию компиляторов GNU, отладчик GNU и другие библиотеки разработки и инструменты, для компиляции ПО.
kernel-package - стема для создания пакетов, связанных с ядром
libncurses-dev - пакет содержит заголовочные файлы, статические библиотеки и символические ссылки, которые понадобятся разработчикам, использующим ncurses.
